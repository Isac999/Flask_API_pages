Coffee Rust Detector
Meu repositório

Objetivo do Projeto:

Construir um modelo em Python que seja capaz de predizer se a folha do café está com a doença da ferrugem (Hemileia Vastatrix) ou saudável. Para isso, utilizei da tecnologia de Deep Learning chamada Redes Neurais Convolucionais (CNN), realizando treinos e testes para a obtenção do melhor modelo. O projeto foi desenvolvido com a linguagem de programação Python e suas respectivas bibliotecas: Tensorflow, Keras, Matplotlib, Flask, Os, Pillow e Numpy. No total foram usadas 16.670 imagens, sendo 11.670 imagens para treino e 5.000 imagens para teste. O resultado foi de uma acurácia de 100% logo no primeiro teste da primeira época.

ATENÇÃO:

O modelo trabalha com predições de imagens com tamanho de 128x128 pixels, então caso a imagem upada seja maior que isso o modelo fará um recorte na imagem. Caso a imagem tenha um tamanho muito grande, é recomendado que a imagem seja upada já recortada com foco na ferrugem. Caso não seja feito o próprio modelo fará o recorte da imagem de forma centralizada para o tamanho de 128x128 pixels.
